### CSS选择器
#### 元素选择器
#### class选择器
- 一个类样式可以作用于多个标签（比如粉红的p和粉红的span）
- 一个标签页可以应用多个类样式（比如p文本既是蓝色的，还有下横杠）
#### id选择器
- id和name的区别，id可以在最短路径下获取指定DOM元素，并且可以快速跳转
#### 群组选择器
- 逗号分割，可以组合任意选择器
#### 通配符选择器
#### 属性选择器
#### 子代选择器 *
- :first-child
- :last-child
- :only-child
#### 同辈选择器
- h2 + p

- 后代选择符
- 伪类，指某元素的某种状态下应用的样式  
  - :hover
  - :link
  - :visited
  - :active
- 为元素，指特定场景的元素 
  - :first-child
  - :first-letter
  - : first-line
  - :focus
  - :before
  - :after
  - ::selection，选中的元素
